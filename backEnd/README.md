## 개요

- **한 줄 요약** : *울반* 프로젝트는 NFC와 BLE 통신을 기반으로 한 학급 소통 앱입니다.

- **기획의도** : 코로나 이후 악화된 교우관개를 개선을 위해 제작되었습니다.

- **개발 인원 및 기간**
  
  - **개발 인원** : Android 3명, BackEnd 3명
  
  - **프로젝트 기간** : 2024.04.08 ~ 2024.05.19

- **주요 기능**
  
  - NFC 기반의 태깅인사, 이어달리기
  
  - 블루투스 기반의 함께달리기
  
  - 학급별 채팅, 게시판, 알림장
  
  - 학급별 소통 통계
